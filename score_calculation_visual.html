🎯 Quiz Score & Percentage Calculation Flow

📋 Level 1: Individual Question Score (Per Answer)

When a team submits an answer to a single question:

Question Marks Answer Correct Answer Is Correct? Marks Obtained
Q1: What is 2+2? 10 4 4 ✓ YES 10
Q2: What is 5+5? 10 10 10 ✓ YES 10
Q3: What is 3×5? 10 14 15 ✗ NO 0
Q4: What is 6×6? 10 - 36 ⊘ SKIP 0
PHP Code (models/QuestionModel.php - saveAnswer):
$isCorrect = ($selectedAnswer === $question['correct_answer']) ? 1 : 0;
$marksObtained = $isCorrect ? $question['marks'] : 0;
✓ If answer matches correct_answer → marks_obtained = question marks
✗ If wrong or skipped → marks_obtained = 0
💾 Stored in: team_answers table
📊 Level 2: Round Total Score (All Questions in One Round)

After team completes all questions in a round, the system sums up all scores:

SQL Calculation:
SELECT
  SUM(marks_obtained) as total_marks,
  SUM(CASE WHEN is_correct = 1 THEN 1 ELSE 0 END) as correct_answers,
  SUM(CASE WHEN is_correct = 0 THEN 1 ELSE 0 END) as wrong_answers,
  COUNT(*) - correct_answers - wrong_answers as skipped_answers
FROM team_answers
WHERE team_id = ? AND round_id = ?

SQL Calculation for Cumulative Score:
SELECT
  SUM(total_marks) as total_marks,
  SUM(correct_answers) as total_correct,
  SUM(wrong_answers) as total_wrong,
  SUM(skipped_answers) as total_skipped,
  COUNT(*) as rounds_completed
FROM round_results
WHERE team_id = ? AND status = 'completed'
Percentage Formula:
Overall_Percentage = (Total Marks Obtained / Total Possible Marks) × 100

In Example: (150 / 200) × 100 = 75%

💾 Stored in: overall_results table (Used for Leaderboard & Dashboard)

🗄️ Database Tables Reference
📋 team_answers
id Primary Key
team_id Which team
question_id Which question
selected_answer A, B, C, or D
is_correct 1 = Correct, 0 = Wrong
marks_obtained Points earned per Q
📊 round_results
id Primary Key
team_id Which team
round_id Which round
total_marks SUM per round
correct_answers COUNT correct
wrong_answers COUNT wrong
percentage % score
🏆 overall_results
id Primary Key
team_id Which team
total_marks SUM all rounds
total_correct SUM all rounds
total_wrong SUM all rounds
rounds_completed Count of rounds
percentage Overall %
🎯 Key Takeaways
1️⃣ Individual Question: marks_obtained = answer_correct ? question_marks : 0
2️⃣ Round Total: SUM of all marks_obtained for that round
3️⃣ Overall Total: SUM of all round totals (cumulative across all rounds)
4️⃣ Percentage: (Total Marks Obtained / Total Possible Marks) × 100
5️⃣ Leaderboard: Ranks teams by overall_results.total_marks (highest first)
